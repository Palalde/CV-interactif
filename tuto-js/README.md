## 💡 Conseils d'apprentissage

### ✅ Bonnes pratiques
1. **Testez tout** dans la console (F12)
2. **Utilisez console.log()** partout
3. **Commencez simple**, ajoutez progressivement
4. **Faites des pauses** régulières
5. **Relisez le cours** si vous bloquez
6. **Expérimentez** avec des variations

### ❌ Erreurs à éviter
1. Copier-coller sans comprendre
2. Vouloir tout parfait du premier coup
3. Abandonner au premier bug
4. Sauter les tests dans la console
5. Demander la solution directe

- [ ] ✅ Le code fonctionne (pas d'erreur console)

### Niveaux de maîtrise
- **⭐ Débutant** : J'ai fait avec tous les indices
- **⭐⭐ Confirmé** : J'ai fait avec 2-3 indices max
- **⭐⭐⭐ Avancé** : J'ai fait avec 1 indice ou sans
